Sequencing coverage calculator

Please check the official Sequencing Coverage Calculator tool to verify your results before finalising the experiment design. We also maintain a list of coverage and read recommendations for our genomics facility users based on scientific publications.

Please check this short video regarding the usage information for the coverage calculator tools.

Metadata reformat

Use the metadata template from download menu and add the required information. Then upload the completed file using the file selector option on Metadata reformatting tool page.

Please check this short video regarding the usage information for the metadata reformatting tools.

Samplesheet reformat

Use the samplesheet templates from download menu and add the required information. Then upload the completed file using the file selector option on Samplesheet reformatting tool page.

Please check this short video regarding the usage information for the samplesheet reformatting tools.

Metadata validation

Upload the completed and reformatted metadata csv file(s) and samplesheet csv file using the file selectors on the Metadata validation page.
